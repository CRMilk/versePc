# VersePC

Minecraft 启动器，支持微软账户/外置登录/离线三种登录方式。

## 功能特性

- 全新 UI 设计，支持亮色/暗色主题
- 微软账户 / 外置登录 / 离线三种登录方式
- Modrinth / CurseForge 模组下载与管理
- 内置 AI 智能助手
- 陶瓦联机多人游戏
- 3D 皮肤预览
- 多源自动更新检测，支持 GitHub 和夸克网盘下载
- Java 环境自动检测与引导安装
- 游戏崩溃自动分析
- 整合包导入支持
